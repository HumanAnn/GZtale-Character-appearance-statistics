GZtale Character Appearance Archive
====================================
This archive contains all pages from the GZtale comic (Anamnesis + Bloodshed) with a convenient tag-based search system for finding characters.

### How to Use Tag Search

1. Download and extract the archive to any folder.
2. Open File Explorer and navigate to either (Comic without folders or Comic with folders).
3. In the search bar (top-right corner), enter one or more tags.

### Search Features

- You can search for multiple tags by separating them with commas.

  EXAMPLE: tags: Ganz, Undyne, Part_3

- Search is not case-sensitive (Ganz = ganz = GANZ)
- You can make tags negative, for example, to see how many pages are without Ganz:

  EXAMPLE: tags: -Ganz

### Archive Structure

- Full_Comic — the complete comic in chronological order.
- Characters — pages sorted by character.

Inside the character folders:

- Anamnesis is divided into: Part 1, Part 2, Part 3, and Part 4.
- Bloodshed is divided into: Chapter 1, Chapter 2, Chapter 3, and Chapter 4.

### Chapter Search

- For Anamnesis, use either: Part1 or Part_1.
- For Bloodshed, use either: Chapter1 or Chapter_1.

Both formats work equally well.

### Examples
- tags: Ganz, Part_3 - Shows all appearances of Ganz in Part 3 of Anamnesis.
- tags: Ganz, Papyrus, Chapter2 - Shows pages where Ganz and Papyrus appear together in Chapter 2 of Bloodshed.

### Additional Information
- Every page includes both English and Russian tags for compatibility with different archive versions.

If you have any questions, feel free to leave a comment under the post.

Have fun exploring! ❤️


На русском
====================================
Это архив страниц комикса GZtale (Anamnesis + Bloodshed) с удобной системой поиска по персонажам.

### Как пользоваться поиском по тегам

1. Откройте архив и распакуйте его в удобную папку.
2. В Проводнике Windows перейдите в нужную папку (Comic without folders или Comic with folders).
3. В строке поиска (в правом верхнем углу) введите:

   теги: Ганз (или любого персонажа, часть или главу)  

Важные возможности поиска:
- Можно искать несколько тегов через запятую:  

  ПРИМЕР: теги: Ганз,Андайн,Часть_3

- Поиск не чувствителен к регистру (Ганз = ганз = ГАНЗ).
- Можно сделать теги отрицательными, например посмотреть сколько страниц без Ганза:

  ПРИМЕР: теги: -Ганз

### Структура архива

- Full_Comic — весь комикс в хронологическом порядке
- Characters — папки по персонажам

Внутри папок персонажей:
- Anamnesis → разделён на: Part 1, Part 2, Part 3, Part 4.
- Bloodshed → разделён на: Chapter 1, Chapter 2, Chapter 3, Chapter 4.

Важный нюанс при поиске глав:
- Для Anamnesis используйте тег Part1 или Part_1 
- Для Bloodshed используйте тег Chapter1 или Chapter_1

Отличее только в пробеле.

ПРИМЕРЫ:
- теги: Ганз, Часть_3 → появления Ганза в 3-й части Anamnesis
- теги: Ганз,Папайрус,Глава2 → Ганз и Папайрус во второй главе Bloodshed.

### Дополнительно
- Все теги прописаны на английском + русском для удобства.

Если у вас возникнут вопросы — пишите в комментариях под постом.
